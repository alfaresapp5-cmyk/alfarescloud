## التقنيات المستخدمة

### Frontend Framework
- **React 18.3.1**: مكتبة JavaScript الحديثة
- **TypeScript**: لكتابة كود آمن ومنظم
- **Vite**: أداة بناء سريعة

### UI Framework
- **Tailwind CSS v4**: تصميم عصري ومرن
- **Radix UI**: مكونات UI قابلة للوصول
- **Material-UI**: مكونات جاهزة احترافية
- **Lucide React**: أيقونات عصرية

### Charts & Visualization
- **Recharts**: رسوم بيانية تفاعلية
- **Motion (Framer Motion)**: حركات وانتقالات سلسة

### Forms & Validation
- **React Hook Form**: إدارة النماذج
- **Date-fns**: معالجة التواريخ

### State Management
- React Context API
- React Hooks

## متطلبات التشغيل

- **Node.js**: الإصدار 18 أو أحدث
- **pnpm**: مدير الحزم (يُفضل)
- **متصفح حديث**: Chrome, Firefox, Safari, Edge

## التثبيت والتشغيل

### 1. تثبيت المكتبات
```bash
pnpm install
```

### 2. تشغيل خادم التطوير
```bash
pnpm dev
```

### 3. بناء المشروع للإنتاج
```bash
pnpm run build
```
